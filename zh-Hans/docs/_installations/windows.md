### Windows

在 Windows 上有2种方式来安装 Yarn。

#### 下载安装程序

你可以在 Windows 上获取 `.msi` 文件, 运行时将引导你完成 Yarn 的安装。

如果你使用安装程序的方式, 你首先需要安装 [Node.js](https://nodejs.org/)。

<a class="btn btn-primary" href="/latest.msi">安装程序下载</a>


#### 通过 Chocolatey 安装
